Using The Calendar

The Calendar feature of OPA

  • is a calendar system implemented as a Java servlet.
  • can be used as a standard calendar system and for booking some resources.
  • supports WAP/WML so it can be used even from mobile phones.
  • can have users notified by e-mail about schedule changes.
  • supports file based persistence - data will be saved in flat files or in a database.
  • supports localization.

Consult your Technical Support team f you wish to use the more sophisticated features of the calendar.
