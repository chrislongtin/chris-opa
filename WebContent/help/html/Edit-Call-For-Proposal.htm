Edit Call For Proposal

 

Steps:

 

1. Select Calls for Proposals from the menu, to view a list of all CFPs

2. Click on the Title of a CFP

3. A page will display all information about the CFP.  Click on Edit this record to modify the information, or delete this record to remove the CFP.

4. Appraisal criteria specific to a CFP can also be added, edited or deleted from this page. 

 

All criteria from another CFP can also be imported.
