Communications Center

The Communications Center page provides several functionalities related to communicating with the public as well as  within a select group of people.

It provides the facility to compose default letters - templates used as the basis for letters to proponents; send notificaion letters to applicants; and view notification history.

OPA has a built-in mailing lists management system. This allows for the maintainance of mailing lists; sending messages to groups or individuals; and managing the messages.

Public and private discussion forums can also be managed. The public discussion facility is a useful tool for policy discussions, for example. Private discussions are limited to a select group of people.

The FAQ facility allows the public to view FAQs, and provides the administratro the means of adding to the list.
