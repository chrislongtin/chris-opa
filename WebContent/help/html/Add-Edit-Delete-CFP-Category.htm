Add, Edit or Delete CFP Category

CFPs can be categorized to aid analysis. For example, an organization may be offering grants for EDUCATION, COMPUTERS, HEALTH. In this case it might help to assign one or more categories to a CFP.

 

Steps:

 

1. Select Calls for Proposals from the menu, to view a list of all CFPs

2. Click on the Title of a CFP

3. A page will display all information about the CFP.  Click on Edit this record to modify the information, or delete this record to remove the CFP.

4. Categories assigned to the CFP are displayed, if any. Select Add  Category to add a new category. Select Edit

nextto the category to edit, or Delete to delete it
