Language Setup

One of the interesting features of OPA is its ability to handle multiple languages.

An OPA coordinator has the ability to setup languages:

Add New Language

View/Modify Phrases

Convert Phrase List 

Add A New Language

  1. Select Language Setup from the menu bar
  2. To add a new language, select Setup a new Language, enter the language in its native form and click Submit

Edit A Language
  1. To modify any phrases for a language, select the appropriate language under View / Modify Phrase List, and click submit
  2. To modify a phrase, click on the Phrase, make your changes, and click Submit.

 Delete A Language

  1. Select Language Setup from the menu bar
  2. Click on the ‘Delete’ link next to the language.

 

Convert Phrases From One Language to Another

  1. Select Language Setup from the menu bar
  2. To setup the website interface in another language, select the originating language and destination language under Setup Phrase List and click submit.
  3. You must translate all phrases, otherwise sections of the website will be missing explanatory text in the new language.
