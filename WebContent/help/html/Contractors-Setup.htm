Contractors Setup

This functionality enables a user with administrative privileges (a coordinator) to:

Add Contractor

Edit/Delete Contractor

Steps:

Select Reviewers/Coordinators/Contractors from the menu bar. A list of existing coordinators, reviewers, and contactors will be displayed.

To add a coordinator, select Add Contractor. 

 

To edit or delete contractor information, select the appropriate link immediately beneath the information you wish to modify.

 

When a contractor is added or edited, the coordinator can choose to associate the contractor to a particular Call For Proposal or to a particular Contracting Agency.

 

No contractor can be deleted if there exists an assignment to a project or proposal and there exists a Timesheet submitted by the contractor pertaining to the project.
