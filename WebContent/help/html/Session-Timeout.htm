Session Time-Out

 

 

Public-site has no login; so sessions are not controlled. Admin-site is session-sensitive. If no activity is carried out for 30 minutes or more, a session times out.  (Consult your network/system administrator if you want to change timeout period to more or less than 30 minutes). All admin-site pages perform a check for session time-out. If the session has timed out, user is required to login first.
