Coordinators Setup

This functionality enables a user with administrative privileges (a coordinator) to:

Add Coordinators

Edit/Delete Coordinators

Steps:

Select Reviewers/Coordinators/Contractors from the menu bar. A list of existing coordinators, reviewers, and contactors will be displayed.

To add a coordinator, select Add Coordinator. 

 

To edit or delete coordinator information, select the appropriate link immediately beneath the information you wish to modify.
