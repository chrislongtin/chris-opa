Professional Skills (Personnel Agency)

Where the CFP is a job placement, candidates that match the SKILLS specified for the job can be automatically selected. To enable this an inventory of professional skills has to be maintained.

Steps:

 

Skills maintenance can be initiated in two ways:

 

Method 1:

 

1. Select Calls for Proposals from the menu, to view a list of all CFPs

2. Click on the Title of any CFP

3. A page will display all information about the CFP.  Click on Professional Skills to to add skills

 

Method Two:

 

Click on Professional Skills from the main menu to to add skills.
