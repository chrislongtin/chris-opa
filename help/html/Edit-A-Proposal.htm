Edit a Proposal

A proposal can be submitted and edited by a proponent (using the Public Website) or registered and edited by an administrator.

Edit A Proposal  (Administrator)

Select Proposal Tracking & Review from the menu bar.

 

Either:

 

Select the Title of the Call for Proposals from the list, and click Search Proposals.

 

A list of all proposals submitted to the CFP will appear. You can limit you selection by using the search facility on the page.

 

Within each proposal you can click on Assign Reviewers to add or remove reviewers assigned to each proposal; or Summary of Reviews / Change Status to view the review information submitted by reviewers and modify the status assigned to the proposal; or View/Modify Proposal Record to edit proponent information or remove the proposal; or Assign Contractors to add or remove contractors assigned to each proposal; or Reviewers Timesheets Management to review and approve timesheets submitted by reviewers; or Contractors Timesheets Management to review and approve timesheets submitted by contractors; or Summary of Proposal Reviews / Change Status to view the review information submitted by reviewers and modify the status assigned to the  reports for the proposal;

 

Click on View/Modify Proposal Record to edit proposal information. You can edit the basic data on the proposal; or Add A Document to the porposal; or Add A Researcher; or change the Status of the proposal. or declare a proposal as

hidden, making it unavailable for public viewing.

 

Edit A Proposal  (Public/Proponent)

Select Edit/Update Your Proposal from the menu bar.

 

Enter the Tracking Code and Password associated with the proposal you wish to edit.

 

Once the tracking code and password are validated, the proposal details are displayed.

You can edit the basic data on the proposal; or Add A Document to the porposal; or Add A Researcher;
