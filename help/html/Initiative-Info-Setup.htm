Initiative Information Setup

 

  • Add Funding Initiative Info
  • Edit Funding Initiative Info
  • Add Funding Agency
  • Edit/Delete Funding Agency

 

    1. To View Initiative Information, select Initiative Information from the menu bar
    2. To edit information, select the language version you wish to edit under Edit Funding Initiative Information, and click on Submit
    3. To add funding initiative information for a new language, either select Compose from the menu bar beneath Initiative Information, or select Add Funding Initiative Information from the Initiative Information page.
    4. From this section you can also setup Funding Agencies information.
    5. Click on Add Funding Agency Information to add information about a new funding agency, or click on Edit Funding Agency Information beneath the funding agency you wish to modify.
