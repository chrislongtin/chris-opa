Proponent Notifications

Proponent Notifications gives the administrator the facilities to setup default messages, send messages and review messages sent to proponents. 

View Notification History

 

This facility allows the asministrator to review messages sent to proponents with regard to a particular Request For Proposal (RFP).

Steps:

    1. Select Communications Center from the menu bar
    2. Select View Notification Letters from the page. A list of messages sent will be displayed.
