Proponent Notifications

Proponent Notifications gives the administrator the facilities to setup default messages, send messages and review messages set to proponents. 

Compose Default Letters

Default messages are used by OPA for correspondence between administration and proponents.

Steps:

  1. Select Communications Center from the menu bar
  2. Select Compose Default Letters from the page. A list of all the default letters that have been set up is displayed. You can set up a new default letter or edit/delete an existing one. Click on subject to view full letter .
