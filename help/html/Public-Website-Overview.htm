Public Website

The Public Website is the facility the public uses to interact with OPA. (The other website is the Admin Website, used to administer OPA).

The public is given the facilities thahe enable them to:

  • Submit Proposals
  • Edit/Update Pproposals
  • Add Resumes (Contractors)
  • Edit Resumes (Contractors)
  • Submit Reports
  • Edit/Update Reports
  • Participate in Public Discussions
  • Search For CFPs
  • View Proposals and Reports

Generally, when a CFP is issued, the link to the public website is part of the package.
