Reviewers Setup

 

This functionality enables a user with administrative privileges (a coordinator) to:

Add Reviewer

Edit/Delete Reviewer

Steps:

Select Reviewers/Coordinators/Contractors from the menu bar. A list of existing coordinators, reviewers, and contactors will be displayed.

To add a reviewer, select Add Reviewer. 

 

To edit or delete reviewer information, select the appropriate link immediately beneath the information you wish to modify.

 

When a reviewer is added or edited, the coordinator can choose to associate the reviewer to particular Calls For Proposal.

 

No reviewer can be deleted if there exists an assignment to a project or proposal and there exists a Timesheet or Review submitted by the reviewer pertaining to the pproposal.
