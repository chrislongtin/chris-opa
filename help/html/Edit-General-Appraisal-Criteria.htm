Edit General Appraisal Criteria

Steps:

    1. Select Appraisal Criteria from the menu bar
    2. A list of all Initiative-wide criteria will appear.  If the review process in active, you will not be able to modify any of the criteria.  Otherwise, you can select edit or delete for each criteria.
    3. To add new criteria, select Compose from the menu bar beneath Appraisal Criteria.
