Add a Document to A Proposal

Supporting or additional documents can be added to a proposal.

On the Public Website, this functionality is accessed through the Edit/Update Proposal link.

On the Administrator  Website, this functionality is accessed through the Proposal Tracking and Review link, followed with the  Edit/Update Proposal link

 Data entry rules are as follows:
