Add a New Language   

 

An OPA coordinator has the ability to add a new language.

 

Steps:

 

  1. Select Language Setup from the menu bar
  2. Select Setup a New Language, enter the language in its native form and click Submit

 

Add A Phrase

1. Select Language Setup from the menu bar

2.  Select Language

3.  Click on View to see the list of Phrases, and the form for entry of new phrases.

4. To add a phrase,  make your entry, and click Submit
