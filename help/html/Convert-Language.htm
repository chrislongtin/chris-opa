Convert Phrases From One Language to Another

An OPA coordinator has the ability to convert phrases from one language to another.

Steps:

 

1. Select Language Setup from the menu bar

 

2. To setup the website interface in another language, select the originating language and destination language under Setup Phrase List and click GO.

 

3. On the next page, enter corresponding translations for the left side phrases on the right hand side. Click Submit.

You must translate all phrases, otherwise sections of the website will be missing explanatory text in the new language.
