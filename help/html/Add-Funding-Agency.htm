Add Funding Agency

As the title 'Funding Agency' implies, this is an organization that is providing the funds on which a program is based. For Grants and Loans, this is an organization that supllies the money to to disbursed. For Job Placements, this is an organiztion that is paying a placement agency for conducting searches to find the right candidates.

1.     Select Initiative Information from the menu bar

2.     Click on Edit Funding Agency Information beneath the funding agency you wish to modify.
