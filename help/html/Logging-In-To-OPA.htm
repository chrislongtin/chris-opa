Logging In To OPA

 

Initial Coordinator User-Id and Password

 

When OPA is installed, it sets up a default Coordinator and a password for the default Coordinator.

The default user id is (usually): admin

The default password is (usually): opajava.

Find out from your system administrator or installer what the actual values are. You need the administrator’s

identifier and password to set up your worksite.

 

Login to OPA

 

Obtain the initial URL for OPA from the system/network administrator or the person who installed OPA.

Enter the URL on the address line of your browser. If everything is okay, the screen that allows you to select what role you want to play appears.

There are four roles: Coordinator, Reviewer, Contractor, and Public (optional).
