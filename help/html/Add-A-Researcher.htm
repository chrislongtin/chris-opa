Add Co-Applicant/Researcher Profile

A researcher or co-applicant can be added to a proposal.

Steps:

Select "Add A Researcher" from the Edit A Proposal page.

The data entry rules are as follows:

 

Data Item

Explanation

Allowed Input

Comments

Researcher – First Name and Initial

The first name of the researcher.

 

First name Required.

Researcher – Last Name

The last name of the researcher.

 

Required

Researcher – Phone

The phone number of the researcher.

 

 

Researcher – Fax

The fax number of the researcher.

 

 

Researcher – Email

The e-mail address of the researcher.

 

Required

Researcher – Citizenship

The county the researcher is a citizen of.

 

Required

Researcher – Country of Residence

The county where the researcher resides.

 

Required

Researcher – Affiliated Institution

The organization the researcher belongs to

 

Required
