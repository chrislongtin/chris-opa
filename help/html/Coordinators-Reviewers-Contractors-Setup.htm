Reviewers,Coordinators and Contractors Setup

This functionality enables a user with administrative privileges (a coordinator) to:

  • Add Coordinator
  • Edit/Delete Coordinator
  • Add Reviewer
  • Edit/Delete Reviewer
  • Add Contractor
  • Edit/Delete Contractor

Steps

  1. Select Reviewers/Coordinators/Contractors from the menu bar.
  2. To add a coordinator, select Add Coordinator.  To add a reviewer, select Add Reviewer. To add a contractor select Add Contractor.
  3. To edit or delete reviewer, contractor, or coordinator information, select the appropriate link immediately beneath the information you wish to modify.
