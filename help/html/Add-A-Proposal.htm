Add a Proposal

A proposal can be submitted by a proponent (using the Public Website) or registered by an administrator.

Register A Proposal  (Administrator)

Select Proposal Tracking & Review from the menu bar.

 

Either:

 

Select the Title of the Call for Proposals from the list, and click Search Proposals.

 

A list of all proposals submitted to the CFP will appear. You can limit you selection by using the search facility on the page.

 

Click on Add a Proposal for a Proponent to add new proposal information for a proponent.

 

Or

 

Select the Title of the Call for Proposals from the list with label Register New Proposal Application, and click Goto Application Form.

 

 

Register A Proposal  (Public/Proponent)

Select Submit A Proposal from the Public Website menu bar.

 

Select the Title of the Call for Proposals from the list.

 

Review eligibility, criteria and proposal submission information.

Click on  Eligibility, criteria and proposal submission information

 

Click on Submit a new proposal now through this website to add new proposal information for a proponent.
