Worksite Setup

 

Edit Worksite

 

It is essential that you set up all the worksite setup information prior to continuing with any other information input.

    1. Select Worksite Setup from the menu at the left
    2. The current site setup criteria will be displayed.  Click on Edit this Record
    3. Fill out all fields on the form, and select Submit at the bottom of the page
    4. The current site setup page will reload with the changes.

 

 

Data Item

Explanation

Allowed Input

Comments

Host URL

This is the web page address that the Coordinator/Administrator will use to access OPA

 

Example: http://www.mycompany.com/opa/admin/

Notice that it has the /admin/ folder.

If you do not have this information,  contact your System or Network Administrator.

If you are using an application server, like TOMCAT, it may be necessary to add a port number (eg 8080, 8090, 8081) after mycompany.com.

Example: http://www.mycompany.com:8080/opa/admin/

Consult your Network Administrator to determine if you need to add port number.

Host Document Directory

This is the directory where image files used by OPA are kept.

This directory also contains all documents submitted by the public (such as proposals, resumes, reports).

“docs”

If you use a different directory name, the program modules that use this directory will have to be modified.

Application Name

This is the name op the application

Commonly ‘opa’

This is related to item #1 (Host URL). In the example

http://www.mycompany.com:8080/opa/admin/

the application name is ‘opa’. In the screen shot above, the application is named “applicants’. This is more specific that just ‘opa’. You can have many OPA applications on the same server. Use distinct names to distinguish them.

Internal Discussion List Name

This is the name of the mailing list you will use for internal discussion.

 

Enter any name. You will have a chance to change it after you set up your discussion list. If you do not plan to use discussion lists, no harm is done by specifying a name.

Proponent information to be shown publicly

How much information about proponents and their proposals do you want to show to make public.

Choices are:Full Information;Accepted Proposals and Reports;No Information

 

Amount of Information to be shown publicly

How much information about proposals do you wan to make public.

Choices are:

  • Complete
  • Limited   (Title/name)

 

Create Multiple CFPs

For this initiative, will you be creating many Requests For Proposals

Put in a check mark if ‘YES’

 

Use Initiative Criteria

When you evaluate proposals, do you wan to apply criteria that apply to the whole initiative in addition to those that are specific to a call for proposal.

Put in a check mark if ‘YES’

 

Use CFP Criteria

When you evaluate proposals, do you wan to apply criteria that apply to the specific call for  proposal.

Put in a check mark if ‘YES’

 

Number of Criteria Rankings

How many choices will an e reviewer have to choose from to assign to a criterion?

Specify an number between 2 and 10

 

Minimum Rank

What is the minimum score for a criterion?

Usually 2

 

Show weight to Public or Reviewers

Do you want the public and/or reviewers to know the weight you assigned to your criteria.
  • Choices are:Both;Reviewers;Public;None

 

Show all reviewer assignments to  each reviewer

Do you want every reviewer to know what other reviewers have been assigned to the same proposals?

Put in a check mark if ‘YES’

 

Reveal review summaries to Reviewers

Do you want reviewers to see the summary of reviews.

Put in a check mark if ‘YES’

 

Background Image File Name

The file name of the background image for the website

Usually a file located in the ‘docs’ directory

Type in full path and name or browse to the directory containing the file. OPA comes with a default file.

Use/Engage Public Interface

Is the public able to access your website.

Put in a check mark if ‘YES’

Under some circumstances, your initiative might be for internal use only. In this case, the public interface of OPA will be disabled. You can re-activate it by editing this page and reversing your selection.

Public Header background File Name

The name of the image file for the public page header

 

Type in full path and name or browse to the directory containing the file. OPA comes with a default file.

Admin Header background File Name

The name of the image file for the Admin page header

 

Type in full path and name or browse to the directory containing the file. OPA comes with a default file.

Allow public to attach documents to messages

Will you allow the public to attach documents when they send discussion messages.

Put in a check mark if ‘YES’

 

Default Language

What language will be the default?

Select one of the languages you have defined.
