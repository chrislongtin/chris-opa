Mailing Lists Management

OPA has a built-in mailing lists management system. This allows for the maintenance of mailing lists; sending messages to groups or individuals; and managing the messages. It also has the ability to link to external mailing lists management systems such as Majordomo, Mailman, DadaMail. Consult your System Administrator if you want to link to external mailing lists managers.

Only administrators/coordinators have the ability to manage mailing lists.

Steps:

  1. Select Communications Center from the menu bar
  2. Select Mailing Lists Management from the page. A list of all the mailing lists that have been set up is displayed. You can set up a new list or perform any of the functions displayed below each list.

Functions Supported

  • Set up New Mailing List
  • Edit List
  • Delete List
  • Members/Memberships (Add, Change, Delete members of lists)
  • Set as default List
  • Send Message
  • View Sent Messages
During Worksite Setup the coordinator/administrator has the option of specifying a default mailing list.
