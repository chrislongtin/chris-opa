Find Candidates (Personnel Agency)

When a Call For Proposal is created, a set of required skill is also recorded. When a candidate/contractor is recorded, the skills possessed by the candidate are recorded.

 

Find Candidates automatically compares these two sets of skills and displays a list of candidates that match.  All the skills for the CFP must be possessed by the selected candidates.

 

Steps:

 

1. Select Calls for Proposals from the menu, to view a list of all CFPs

2. Click on the Title of a CFP

3. A page will display all information about the CFP.  Click on Find Candidates to find candidates that match the skills criteria established for the CFP.
